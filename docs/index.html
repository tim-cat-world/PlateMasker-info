プライバシーポリシー

tim-cat（以下、「当方」）は、Androidアプリ「PlateMasker」（以下、「本アプリ」）におけるユーザー情報の取り扱いについて、以下の通りポリシーを定めます。
【アプリの概要と目的】
本アプリは、AI（YOLO11）を用いて画像からナンバープレート等の情報を自動認識する機能を提供します。本機能はすべて端末内で完結し、ユーザーのプライバシーを最優先に設計されています。

1. AI推論処理と画像の取り扱い

本アプリは、YOLO11アルゴリズムを用いた画像解析（ナンバープレート認識等）を行いますが、以下の事項を厳守しています。

  • 完全ローカル処理： ナンバープレートの認識を含むすべてのAI推論処理は、ユーザーの端末内（オフライン）で実行されます。
  • 外部送信の禁止： 選択された画像が、外部サーバーや第三者へ送信されることは一切ありません。
  • ナンバープレート情報読み取りの禁止： 認識されたナンバープレートからは形状と色情報のみ使用し文字情報は読み取らず、外部サーバーや第三者へ送信されることは一切ありません。
  • 非保存の原則： 認識した情報は一時的な処理にのみ使用され、アプリ内や端末内に永続的に保存されることはありません。

2. 収集する情報と利用目的

本アプリでは、広告配信のために以下の第三者提供のSDKを使用しており、これらがユーザー情報を自動的に取得する場合があります。

  • Google AdMob (Google LLC): 広告配信およびパーソナライズ、広告効果の測定のために、広告識別子（IDFA/AAID）、デバイス情報、位置情報（概略）が収集される場合があります。

3. 第三者への提供

当方が直接ユーザーの個人情報を収集し、第三者に提供することはありません。ただし、前述の広告配信プラットフォームが収集するデータについては、各社のプライバシーポリシーに基づいて管理されます。

4. データの保持と削除

当方はユーザーの個人情報を直接サーバー等に保存しません。広告用識別子のリセットについては、ユーザーご自身が端末の設定から行うことができます。

5. 免責事項

本アプリの利用により生じた損害について、当方は一切の責任を負いません。本アプリの内容、広告から遷移した先のサイトの内容については、それぞれの提供元へお問い合わせください。
